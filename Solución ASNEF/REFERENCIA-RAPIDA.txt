# Archivo de Referencia Rápida - ASNEF
# Úsalo como checklist mientras configuras

## CREDENCIALES QUE NECESITAS

### 1. De Supabase (después de crear proyecto)
```
supabaseUrl: https://[project-id].supabase.co
supabaseAnonKey: eyJhbGciOi...
```

### 2. De tu dominio
```
Dominio: www.tudominio.com
Email real: tu@email.com
Teléfono: +34 XXX XX XX XX
NIF/CIF: xxxxxxxxxx
```

### 3. De Formspree (OPCIONAL)
```
FormspreeEndpoint: https://formspree.io/f/[id]
```

---

## ARCHIVOS A COMPLETAR ANTES DE PUBLICAR

Abre cada archivo en VS Code y rellena los campos con ❌

### auth-config.js
```javascript
window.AUTH_CONFIG = {
  supabaseUrl: '', ❌ REQUERIDO
  supabaseAnonKey: '', ❌ REQUERIDO
  // resto igual
};
```

### site-config.js
```javascript
window.SITE_CONFIG = {
  siteUrl: '', ❌ REQUERIDO - tu dominio
  publicContactEmail: '', ❌ REQUERIDO
  publicContactPhone: '', ❌ REQUERIDO
  ownerName: '', ❌ REQUERIDO
  taxId: '', ❌ REQUERIDO
  legalAddress: '', ❌ REQUERIDO
  // lo demás puede quedar vacío o con defaults
};
```

### lead-config.js (OPCIONAL)
```javascript
window.LEAD_CONFIG = {
  endpoint: '', // Tu endpoint de Formspree o similar
};
```

1. ✅ Crear proyecto en Supabase

5. ✅ Completar site-config.js
6. ✅ Registrarte en tu sitio
7. ✅ Convertirte en admin (SQL INSERT)

11. ✅ Configurar dominio DNS

---

## SI ALGO FALLA

Abre navegador > F12 > Console > Copia el error rojo

Errores comunes:
- "Cannot read properties of undefined 'supabaseUrl'" → auth-config.js mal completado
- "RLS policy violation" → Usuario no tiene permisos en Supabase
- "Network error" → Supabase URL incorrecta o sin acceso a internet
